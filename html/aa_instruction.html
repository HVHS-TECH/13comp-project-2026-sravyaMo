INSTRUCTIONS

Welcome to Alien Annihilator! here are all the rules 
    1. You start with 3 lives.
    2. Shoot as many aliens as possible
    3. If an alien or an asteroid hits you, you lose a live
    4. Once all lives are lost the game ends
    5. You can only shoot one laser at a time
    6. Once the game is over, simply click the start button to play again
